Project Overview

XDG (Accelerated Discretized Geometry) is a high-performance geometry acceleration system for OpenMC that significantly improves particle transport simulation performance through advanced discretization techniques. This project focuses on shared memory parallelism due to the higher memory footprint of the geometry representation employed by XDG.
